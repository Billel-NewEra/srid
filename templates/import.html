Import Excel

Importer un fichier

Importez vos fichiers Excel existants. Le système détectera automatiquement le type d'opération selon le nom de la feuille.

Instructions

Détection automatique

Le type d'opération est détecté selon le nom de la feuille Excel (chèque, virement, versement).

Format attendu

La première ligne doit contenir les en-têtes. Les données commencent à la ligne 2.

Colonnes détectées

Date, Client, Montant et Banque sont détectés automatiquement dans chaque ligne.

Note

Les lignes sans date ou sans montant sont ignorées. Vous pouvez importer le même fichier plusieurs fois (les doublons ne sont pas détectés).
